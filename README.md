## 📖 About

DevReview is a full-stack web application where developers can showcase their projects, receive honest reviews, interact with the community, and discover amazing work from other developers.

The platform is designed with a clean, minimal, and premium developer-focused UI inspired by modern SaaS applications.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/devreview.git
```

---

## Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

---

## Environment Variables

Create a `.env` file inside the backend.

```env
PORT=

MONGODB_URI=

JWT_SECRET=

EMAIL=

EMAIL_PASSWORD=

FRONTEND_URL=
```

---

## Run Backend

```bash
npm run dev
```

---

## Run Frontend

```bash
npm run dev
```
